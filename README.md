# GraphQL Practice Lab

이 레포지토리는 GraphQL 서버를 직접 구축하고 다양한 기능을 테스트하며 학습하기 위한 공간입니다.

- **GraphQL 기본 개념 이해**: 스키마(Schema), 타입(Type), 필드(Field) 정의
- **서버 구축**: Node.js 환경에서 GraphQL 서버 셋업
- **TODO**:
  - **Query**: 데이터 조회
  - **Mutation**: 데이터 생성, 수정, 삭제
  - **Subscription**: 실시간 데이터 처리 (옵션)
- **Resolver 패턴**: 데이터 소스와 GraphQL 스키마 연결

## 🛠️ 기술 스택

- **Runtime**: Node.js
- **Language**: TypeScript
- **Library**: Apollo Server

## 🚀 시작하기

프로젝트 초기 설정 후 아래 명령어로 실행할 수 있습니다. (추후 업데이트 예정)

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
